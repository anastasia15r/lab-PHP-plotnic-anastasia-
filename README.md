# 3. Работа с датой и массивами

Создаём массив месяцев для корректного отображения даты на русском:

```php
$months = [
    1 => 'января', 'февраля', 'марта', 'апреля', 'мая', 'июня',
    'июля', 'августа', 'сентября', 'октября', 'ноября', 'декабря'

Определяем текущую дату и день недели:

```php
$current_date = date('d') . ' ' . $months[date('n')] . ' ' . date('Y') . ' года';
$today = date("N"); // 1 = понедельник, 7 = воскресенье
```

Создаём массивы рабочих дней сотрудников:

```php
$john_working_days = [1, 3, 5]; // Пн, Ср, Пт

# 4. Формирование таблицы сотрудников

Используем функцию `in_array()` и тернарный оператор для определения рабочего дня:

```php
<table border="1">
<tr>
<th>№</th>
<th>Сотрудник</th>
<th>График</th>
</tr>

<tr>

</tr>

```

**Объяснение кода:**

* `in_array($today, $john_working_days)` проверяет, есть ли текущий день в массиве рабочих дней сотрудника.
* Тернарный оператор `? :` позволяет выбрать одно из двух значений: если день рабочий — выводим график, иначе — «Нерабочий день».

# 5. Цикл FOR

Пример цикла `for` с выводом промежуточных значений:

```php
$a = 0;
$b = 0;

for ($i = 0; $i <= 5; $i++) {
    $a += 10;
    $b += 5;
    echo "Шаг $i : a = $a , b = $b<br>";
}

echo "<br>Итог: a = $a , b = $b";
```

**Объяснение:**

* `$i` — счётчик цикла, который проходит от 0 до 5.
* На каждом шаге `$a` увеличивается на 10, `$b` — на 5.
* После цикла выводятся итоговые значения.

---

# 6. Цикл WHILE

Переписываем ту же логику с помощью `while`:

```php
$a = 0;
$b = 0;
$i = 0;

while ($i <= 5) {
    $a += 10;
    $b += 5;
    echo "Шаг $i : a = $a , b = $b<br>";
    $i++;
}

echo "<br>Итог: a = $a , b = $b";
```

**Объяснение:**

* Цикл выполняется до тех пор, пока условие `$i <= 5` истинно.
* Проверка условия происходит **перед каждой итерацией**.

---

# 7. Цикл DO-WHILE

Цикл, который выполняется минимум один раз:

```php
$a = 0;
$b = 0;
$i = 0;

do {
    $a += 10;
    $b += 5;
    echo "Шаг $i : a = $a , b = $b<br>";
    $i++;
} while ($i <= 5);

echo "<br>Итог: a = $a , b = $b";
```

**Объяснение:**

* Тело цикла выполняется **сначала**, затем проверяется условие.
* Даже если условие изначально ложное, шаг выполнится один раз.

---

# 8. Контрольные вопросы

1. **Разница между циклами:**

   * `for` — используется при известном количестве повторений.
   * `while` — выполняется, пока условие истинно.
   * `do-while` — выполняется минимум один раз, проверка условия после выполнения тела.

2. **Тернарный оператор:**
   Сокращённая форма `if-else`:

   ```php
   условие ? значение1 : значение2;
   ```

3. **Что произойдет если условие do-while изначально ложное:**
   Тело цикла всё равно выполнится один раз.
